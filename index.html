ezzybot

EzzyBot is a IRC bot framework built in python.

import ezzybot

mybot = ezzybot.bot()

mybot.run({"nick": "MyEzzyBot", "channels":["#ezzybot"], "host": "irc.freenode.net"})

Installation

pip install ezzybot

Or, install the development version

pip install git+https://github.com/ezzybot/ezzybot
